Gridsync URI Scheme
===================

Generic syntax
--------------

``gridsync://<introducer_key>@<introducer_address>:<introducer_port>/<cap>``


Valid queries
-------------


i = introducer (in format <key>@<address>:<port>; "pb://" prefix and "/introducer" suffix are not required)

s = storage server

c = cap

f = name

st, n = shares.total

sn, k = shares.needed

sh, h = shares.happy
